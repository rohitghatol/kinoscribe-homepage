Learn

KinoScribe is a 3D Presentation creation tool. It is based on HTML5 Web Components. Hence it is super easy to use. All you have to do is to import the Web Component on your page and start using it.

More over KinoScribe is meant to be extensible. If you have a great idea about an Cool Transition or Layout or Any Tools around Presentation, go ahead and write your own Custom Element. Your Custom Element or JavaScript code can interact with KinoScribe using Attributes and Events

Basics

Step 1

Include platform.js script and import kinoscribe web component

Step 2

Use <kino-pres> as presentation tag and use <kino-slide> as slide tag

Attributes

<kino-pres> Attributes

Attribute Default Value Description
currentSlide 1 Read currentSlide to know the currently shown slide.Change currentSlide to transition to another slide
slides Read slides to know total number of slides in the kino-pres element.
transition Overrides kino-slide's attributes with 3d layout provided by given transition.

Requires user to include corresponding kino-transition web component on the page. That is if user wants to assign transition to "kino-transition-box", there needs to be an element named <kino-transition-box/>.

Following are options for transitions -
kino-transition-left-to-right Left to Right Transition
kino-transition-left-to-right-emphasis Left to Right Transition with 3D Effect
kino-transition-top-to-bottom Left to Right Transition
kino-transition-top-to-bottom-emphasis Left to Right Transition with 3D Effect
kino-transition-box Transitions as a 3D Box
kino-transition-3d-spiral Transitions as a 3D Spiral
kino-transition-file-cabinet Transitions as a 3D file cabinet


Note - You can create and add your own custom transition.

<kino-slide> Attributes

Attribute Default Value Description
x 0 X Position of Slide
y 0 Y Position of Slide
z 0 Z Position of Slide
rotateX 0 Rotation along X Axis
rotateY 0 Rotation along X Axis
rotationZ 0 Rotation along X Axis
scale 1 zoom in (positive value) or zoom out (negative value) levels
